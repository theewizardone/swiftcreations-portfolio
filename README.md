Swift Creations Portfolio

This repository contains the source code for the Swift Creations agency portfolio website.

This is a single-page, responsive marketing site built using plain HTML and Tailwind CSS.

Structure

The project is structured as a single file, which makes it ideal for immediate deployment to static hosting services like GitHub Pages, Vercel, or Netlify.

index.html: The main and only file containing all HTML, CSS (via Tailwind), and JavaScript code.

README.md: This file.

.gitignore: Basic file to ignore common local project files.

Deployment

To deploy this site using GitHub Pages:

Push this entire folder to a new public GitHub repository.

Go to the repository Settings -> Pages.

Select the main branch (or your primary branch) and the / (root) folder as the source.

Save. Your site should be live within minutes
